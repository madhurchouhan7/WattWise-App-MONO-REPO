# Milestone v1.0 Requirements

## Plan Activation Routing (NAV)
- [ ] **NAV-01**: Upon activating the AI Plan from `PlanReadyScreen`, the user must be redirected to `ActivePlanScreen`.
- [ ] **NAV-02**: The Bottom Navigation Bar must remain visible when viewing the `ActivePlanScreen` (i.e. do not push replacement on top of root).
- [ ] **NAV-03**: When a new plan is created, it is properly activated/viewable in `ActivePlanScreen` rather than dropping back to `DesignPlanScreen`.

## Cache Integrity & Plan Deletion (CACHE)
- [ ] **CACHE-01**: When the user explicitly clicks Delete Plan in `ActivePlanScreen`, the App destroys the local cache.
- [ ] **CACHE-02**: Tapping the back arrow or navigating away does not accidentally corrupt local cache.
