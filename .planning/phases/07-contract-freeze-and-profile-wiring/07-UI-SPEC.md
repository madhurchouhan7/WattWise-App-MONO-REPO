## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Inline icon/avatar border padding, micro spacing |
| sm | 8px | Tight button/icon inset and compact vertical rhythm |
| md | 16px | Default field and row spacing |
| lg | 24px | Section spacing between profile blocks |
| xl | 32px | Primary section breaks on profile screen |
| 2xl | 48px | Major grouping separation in long forms |
| 3xl | 64px | Page-level top/bottom rhythm on utility screens |

Exceptions: Ensure minimum touch target size is 44px for icon-only hit areas (applies to profile avatar edit affordance and retry affordances).

---

## Typography

| Role | Size | Weight | Line Height |
|------|------|--------|-------------|
| Body | 14px | 400 | 1.5 |
| Label | 12px | 600 | 1.3 |
| Heading | 22px | 600 | 1.2 |
| Display | 28px | 600 | 1.2 |

Notes: Phase 7 contract limits text weights to 400 and 600 to reduce visual inconsistency during profile state handling and validation messaging.

---

## Color

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | #F8FAFC | Page backgrounds and neutral section canvases |
| Secondary (30%) | #FFFFFF | Cards, grouped menu/list surfaces, dialogs |
| Accent (10%) | #2563EB | Primary profile actions, active profile affordances, retry action emphasis |
| Destructive | #EF4444 | Destructive actions only (sign out confirmation action, destructive text/icons) |

Accent reserved for: Save Profile primary CTA, inline retry actions for failed profile load/update, active profile edit indicators (avatar edit ring/icon), and focused input state indicators.
